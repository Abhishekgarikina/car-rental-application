# 🚗 Car Rental Web Application

## 📌 Overview
This project is a professional frontend-based Car Rental Web Application developed using HTML, CSS, and JavaScript. It simulates a real-world car rental platform with user authentication, car browsing, booking functionality, and booking history management.

---

## ✨ Features

### 🔐 Authentication System
- User, Admin, and Owner login roles
- Signup with email and password
- Role-based access simulation
- Data stored using browser localStorage

---

### 🚗 Car Management
- Multiple cars with detailed specifications
- Categories (SUV, Sedan, Hatchback, etc.)
- Price per day
- Fuel type display

---

### 🔍 Search & Filter
- Search cars dynamically
- Instant filtering based on user input

---

### 📅 Booking System
- Select start and end date
- Automatic price calculation
- Booking confirmation system
- Simulated email confirmation

---

### 📜 Booking History
- View all previous bookings
- Shows:
  - Car name
  - Booking dates
  - Total price
  - Status

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage (for data persistence)
